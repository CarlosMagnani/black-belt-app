# BlackBelt

Management platform for Brazilian Jiu-Jitsu academies. Connects owners, professors, and students around class attendance, belt progression, and membership.

## Language

**Academy**:
A martial arts school registered on the platform. Created by an owner, joined by students via invite code.
_Avoid_: School, dojo, gym

**Owner**:
The user who created and manages an academy. One owner per academy, one academy per owner.
_Avoid_: Dono, admin, manager

**Professor**:
An instructor assigned to an academy by the owner. Validates check-ins and manages student belts.
_Avoid_: Instrutor, coach, teacher

**Student**:
An academy member who trains and attends classes. Joins via invite code. Cannot self-assign belt or approve check-ins.
_Avoid_: Aluno, member, trainee

**Invite Code**:
A unique code in the format `BB-XXXXXX` generated by the system when an academy is created. Required for any student to join an academy.
_Avoid_: Join code, access code, registration code

**Belt**:
A student's rank in the BJJ hierarchy: white, blue, purple, brown, black, coral, red. Controlled exclusively by owners and professors.
_Avoid_: Rank, grade, level

**Degree**:
A stripe on a belt, numbered 0-4. 24 approved check-ins advance one degree.
_Avoid_: Stripe, grau, rank

**Check-in**:
A student's attendance request for a specific class. Always starts as pending; requires owner or professor approval to count toward belt progression.
_Avoid_: Attendance, presence, presença

**Membership Plan**:
A payment option defined by the academy owner (name, price, period). Does not process payments in MVP.
_Avoid_: Subscription, package, plano

**Membership Subscription**:
A student's link to a specific membership plan within an academy. Status is managed manually by the owner.
_Avoid_: Membership, plano do aluno

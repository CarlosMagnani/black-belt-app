# Prompt: Create a Handoff

Use this at the end of any AI session to update the handoff file.

---

## Template

```
I just finished a working session on BlackBelt — a BJJ academy management platform.

Session summary:

Goal of this session: [what we were trying to do]

What was completed:
- [item 1]
- [item 2]

What was NOT completed:
- [item — reason]

Files changed:
- [path]: [what changed]

Decisions made:
- [decision]: [reason]

Commands run that the next session needs to know:
- [command]: [what it does]

Open questions (need human input):
- [question]

Risks identified:
- [risk]

What should happen next (in order):
1. [first step]
2. [second step]

Please write a complete update to docs/ai-context/current-handoff.md using the template in that file.
The output should be ready to paste directly into the file.
```
